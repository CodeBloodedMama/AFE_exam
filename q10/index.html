Spørgsmål 10 - Native web component

Forklar hvad en native web component er:

En native web component er en brugerdefineret HTML-element, der er bygget ved hjælp af standard webteknologier som HTML, CSS og JavaScript. Disse komponenter kan genbruges på tværs af forskellige projekter og browsere uden behov for eksterne biblioteker.

Hvilken funktionalitet kræves af browseren:

For at understøtte native webkomponenter skal browseren understøtte følgende teknologier:

  • Custom Elements: Tillader oprettelse af brugerdefinerede HTML-elementer.
  • Shadow DOM: Giver mulighed for at kapsle en komponents stil og struktur.
  • HTML Templates: Tillader oprettelse af genanvendelige HTML-skabeloner.
